# ShopBackground_Go
### 项目介绍
开发一套食品商铺的后台管理系统。


### 项目架构
* 前端：Vue框架来进行页面功能实现。
* 后端：Iris框架进行实现，使用Iris框架开发服务器后台功能。
* 数据库：使用的数据库**mysql**和**redis**。关系型数据库使用mysql对表数据进行持久化；对于项目中的一些变动频率不高的数据，选择非关系型数据库redis：来进行存储。
* orm：数据库对象映射框架采用**xorm**进行实现。
